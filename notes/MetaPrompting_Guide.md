## What Is Meta-Prompting?

Instead of writing a prompt yourself, you ask Copilot to write the prompt for you — by telling it your role, your task, and what you need.

You are essentially saying:

> *"You are a prompt expert. You know how to talk to AI. Write me a prompt I can use for my actual work."*

The output becomes your **reference prompt** — something you refine and reuse across different HR workflows and employee lifecycle situations.

---

## The Core Meta-Prompt Formula

```
Act as a Microsoft Copilot prompt expert.
I am a [YOUR ROLE] at [COMPANY/CONTEXT].
I need to [WHAT YOU WANT TO DO].
My audience is [WHO WILL READ THE OUTPUT].
Write me a detailed, structured Copilot prompt I can use to complete this task.
The prompt should specify: role, task, context, output format, and tone.
```

### 1. Employee Onboarding Communication

```
Act as a Microsoft Copilot prompt expert.
I am an HR Business Partner at Bajaj Finance Limited, an NBFC. I receive new joiner details —
name, designation, business unit, reporting manager, and start date — from the Talent Acquisition
team. I need to produce a structured onboarding welcome pack: a day-one welcome email for the
new employee, a checklist email for the hiring manager, and a brief announcement for the team.

My audience includes the new joiner, their manager, and their immediate team — all internal
stakeholders.

Write me a detailed, structured Copilot prompt I can use to produce this onboarding communication
set. The prompt should specify: my role, the task, what input I will provide, the exact output
format for each document, and the appropriate warm yet professional tone.
```

### 2. Performance Review Summarisation

```
Act as a Microsoft Copilot prompt expert.
I am an HR Operations Analyst at Bajaj Finance Limited. I collect raw self-appraisal notes and
manager ratings from the performance management system for a review cycle. I need to convert these
raw inputs into a structured Performance Summary for each employee — highlighting key achievements,
development areas, and an overall performance narrative.

My audience is HR leadership and department heads who will use these summaries for calibration
sessions.

Write me a Copilot prompt I can use to produce this performance summary. The prompt should tell
Copilot exactly what to retain, how to handle conflicting ratings, and what tone to use (balanced,
developmental, HR-ready).
```

### 3. HR Policy Q&A and Employee Communication

```
Act as a Microsoft Copilot prompt expert.
I am an HR Shared Services Analyst at Bajaj Finance Limited. I frequently receive employee queries
about HR policies — leave entitlements, reimbursement rules, transfer processes, and POSH
guidelines. I need to draft clear, empathetic, policy-accurate responses to these queries for
circulation via email or the HR helpdesk portal.

My audience is the employee asking the question — non-HR, non-technical, seeking clarity.

Write me a structured Copilot prompt that will produce a policy response from raw query text and
a reference policy excerpt. The prompt should specify format (plain language, 3–5 sentences), tone
(helpful, empathetic, authoritative), and how to handle ambiguous queries.
```

### 4. Training Needs Analysis Summary

```
Act as a Microsoft Copilot prompt expert.
I am an L&D Manager at Bajaj Finance Limited. After completing a Training Needs Analysis survey
across business units, I have raw data — survey responses, skill gap ratings, and manager inputs.
I need to produce a structured TNA Summary Report for HR leadership — grouping gaps by function,
highlighting critical skill deficits, and recommending priority training interventions.

My audience is the CHRO and L&D leadership who will use this to finalise the annual training
calendar.

Write me a Copilot prompt that will produce a TNA Summary from this raw data. The prompt should
specify format (executive summary + function-wise table + recommendations), tone (analytical,
strategic), and what signals to surface (critical gaps, high-priority cohorts, quick-win
opportunities).
```

### 5. Exit Interview Analysis and Attrition Reporting

```
Act as a Microsoft Copilot prompt expert.
I am an HR Analytics Analyst at Bajaj Finance Limited. I collect exit interview responses each
quarter from departing employees across functions. I need to analyse these responses and produce
an Attrition Insight Report — identifying top reasons for exit, flagging high-risk functions or
roles, and surfacing retention recommendations for leadership.

My audience is HR leadership and Business Heads who need actionable insight — not raw verbatim
quotes.

Write me a Copilot prompt that will produce this attrition report from raw exit interview data.
The prompt should specify format (summary paragraph + themed table + recommendations), tone
(objective, data-driven), and what patterns to prioritise (recurring themes, leadership concerns,
compensation).
```

## Still Confused? Run This.

If the prompt Copilot wrote doesn't feel right — too generic, wrong format, wrong tone — don't rewrite it yourself. Use this follow-up prompt instead:

```
The prompt you wrote is a good start, but I need you to adjust it.
Here is what I want to change:

- [Describe what felt off — e.g., "the format doesn't match what my CHRO expects"]
- [Describe what's missing — e.g., "it doesn't tell Copilot to flag high-attrition roles"]
- [Describe the tone issue — e.g., "it sounds too clinical for an employee-facing document"]

Please rewrite the prompt with these corrections.
Keep the same structure (Role / Task / Context / Format / Tone) but fix the parts I've described.
```

## What to Do With the Output

Once Copilot gives you a well-structured prompt:

1. **Read it** — does it reflect your actual HR role and employee context?
2. **Replace the placeholders** — swap generic text with your real employee or policy data.
3. **Run it** — use it on the exercise dataset provided during training.
4. **Save it** — this is your reusable reference prompt for that HR task type.
